Análisis de Datos con MySQL y SQL Server – Proyecto de Consultas y Exploración de Bases de Datos

1. Descripción General

¡Hola!
En este proyecto asumo el rol de analista de datos con el objetivo de practicar y demostrar mis habilidades en consultas SQL, modelado de datos y análisis de información relacional utilizando dos entornos distintos como Visual Studio y MySQL o SQL Server Management Studio (SSMS) con la base de datos AdventureWorks2017.

El propósito principal es aplicar técnicas de creación, manipulación y consulta de bases de datos en ambos entornos, mostrando la capacidad de construir estructuras de datos, ejecutar consultas de diferentes niveles de complejidad y obtener información relevante para el análisis.

2. Estructura del Proyecto

Tenemos 2 archivos los cuales se dividen en:

2.1. Análisis con MySQL (Visual Studio)

- En este primer bloque realizo la conexión y análisis directamente desde Visual Studio, utilizando una conexión con mi servidor MySQL local.
- Configuración de la conexión con el host local.
- Creación del esquema y las tablas necesarias.
- Carga de datos en las tablas creadas.
- Ejecución de consultas en tres niveles de complejidad

2.2 Consultas en SQL Server Management Studio (SSMS)

En esta segunda parte del proyecto realizo una serie de cinco consultas analíticas sobre la base de datos AdventureWorks2017.
Cada consulta está orientada a resolver un objetivo concreto y cuenta con su correspondiente query.

3. Flujo de Trabajo del Proyecto

- Conexión de Visual Studio con el servidor MySQL (host local).
- Creación del esquema y las tablas de trabajo.
- Inserción y validación de los datos cargados.
- Ejecución de consultas SQL en tres fases de complejidad.
- Realización de consultas analíticas en SQL Server Management Studio.
- Análisis de los resultados y conclusiones.

4. Tecnologías y Librerías Utilizadas

SQL & MySQL

5. Habilidades Demostradas

- Conexión y configuración de entornos de bases de datos locales.
- Creación, modelado y carga de tablas relacionales.
- Ejecución de consultas SQL con distintos niveles de complejidad.
- Manejo de funciones agregadas, joins, subconsultas y agrupaciones.
- Análisis de modelos de datos empresariales (AdventureWorks2017).
